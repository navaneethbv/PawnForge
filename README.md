# PawnForge

PawnForge is a modern chess analysis and training platform built from scratch.

## Features

- Interactive drag-and-drop chessboard
- Position analysis with a fast static evaluator
- FEN copy + board utilities (flip/reset)
- Puzzle training mode with hints and validation
- Responsive dark UI

## Getting started

```bash
npm run serve
```

Then open http://localhost:4173.
